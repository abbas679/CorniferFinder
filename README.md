# Cornifer, The Finder

A fast file search tool with a TUI, inspired by Hollow Knight's Cornifer.

## Build Instructions

1. **Dependencies**:
   - C++17-compatible compiler (GCC 8+, Clang 7+, MSVC 19.29+)
   - CMake 3.15+

2. **Build**:
   ```bash
   mkdir build
   cd build
   cmake ..
   cmake --build .
